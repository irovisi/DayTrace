# DayTrace

Программа для отслеживания активности за компьютером на Windows.

![Latest release](https://img.shields.io/github/v/release/irovisi/DayTrace)
![Platform](https://img.shields.io/badge/platform-Windows-blue)

## Возможности

- Отслеживание активности по категориям и приложениям
- Уведомления при превышении лимита
- Лимиты на отдельные категории и приложения
- Экспорт статистики происходит в Obsidian, в папку Time

## Установка

1. Перейди в раздел [Releases](https://github.com/irovisi/DayTrace/releases)
2. Скачай архив последней версии
3. Распакуй архив в удобную папку
4. Запусти `autostart.bat` от имени админитратора, если хочешь чтобы работал режим фокусировки внимания

## Требования

- Windows 10 или Windows 11
