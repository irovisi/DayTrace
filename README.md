# DayTrace

Программа для отслеживания активности за компьютером на Windows.

![Latest release](https://img.shields.io/github/v/release/irovisi/DayTrace)
![Platform](https://img.shields.io/badge/platform-Windows-blue)

## Возможности

- Отслеживание активности по категориям и приложениям
- Уведомления при превышении лимита
- Лимиты на отдельные категории и приложения
